## [Score Semantics](https://virtualpiano.net/virtual-music-education/)

|symbol|meaning|
|---|---|
|.|note|
|. .|short pause|
|.&#124;.|long pause|
|[...]|play notes together simultaneously|
|[. . .]|play the sequence at fastest possible speed|
